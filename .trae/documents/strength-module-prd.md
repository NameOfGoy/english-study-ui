## 1. 产品概述
强化模块是针对薄弱单词进行专注强化的学习模式。通过渐进式展示方式，先让用户根据图片或中文释义回忆单词，再展示完整信息，帮助用户巩固记忆薄弱词汇。

## 2. 核心功能

### 2.1 功能模块
强化模块包含以下核心页面：
1. **强化练习页**：展示单词卡片，支持渐进式学习和记忆反馈

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 强化练习页 | 单词卡片展示 | 显示单词图片、中文释义、音标、例句等信息 |
| 强化练习页 | 记忆状态选择 | 在初始阶段显示"记得"和"不记得"按钮 |
| 强化练习页 | 完成强化 | 选择记忆状态后展示完整信息，显示"下一个"按钮 |
| 强化练习页 | 进度指示 | 显示当前学习进度和剩余单词数量 |
| 强化练习页 | 图片轮播 | 支持多张图片的左右切换和手势滑动 |

## 3. 核心流程

### 用户学习流程
1. 进入强化模式，系统加载待强化单词列表
2. 初始阶段：仅展示图片或中文释义
3. 用户根据展示内容选择"记得"或"不记得"
4. 系统记录选择并展示单词完整信息（音标、例句等）
5. 用户点击"下一个"进入下一个单词
6. 重复流程直到完成所有单词

```mermaid
graph TD
    A[加载强化列表] --> B[展示图片/中文释义]
    B --> C{用户选择}
    C -->|记得| D[调用完成强化接口 operation=1]
    C -->|不记得| E[调用完成强化接口 operation=2]
    D --> F[展示完整信息]
    E --> F
    F --> G[显示"下一个"按钮]
    G --> H{是否还有单词}
    H -->|是| B
    H -->|否| I[完成强化]
```

## 4. 用户界面设计

### 4.1 设计规范
- **主色调**：深橙色（与强化状态对应）
- **按钮样式**：圆角矩形，采用3D立体效果
- **字体规范**：
  - 标题：18px 粗体
  - 正文：14px 常规
  - 辅助信息：12px 细体
- **布局风格**：卡片式布局，居中显示
- **图标风格**：使用圆润线条图标

### 4.2 页面布局

| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 强化练习页 | 单词卡片 | 圆角卡片，白色背景，阴影效果，包含图片区域和文字区域 |
| 强化练习页 | 图片轮播 | 图片居中显示，底部指示器，左右切换按钮 |
| 强化练习页 | 按钮组 | "记得"（绿色）、"不记得"（红色）、"下一个"（主色调） |
| 强化练习页 | 进度条 | 顶部进度条，显示完成百分比 |

### 4.3 响应式设计
- **桌面优先**：默认适配桌面端，支持鼠标操作
- **移动端适配**：支持触摸滑动操作
- **按钮大小**：移动端按钮更大，便于触摸操作

## 5. 技术实现要点

### 5.1 状态管理
- `strengthState`: 'choice' | 'full'  // 选择阶段 vs 完整展示阶段
- `currentWordIndex`: 当前单词索引
- `wordList`: 强化单词列表
- `loading`: 加载状态
- `error`: 错误信息

### 5.2 接口调用
- 获取强化列表：`getStrengthWordCardList()`
- 完成强化：`finishStrength({ wordId, operation })`

### 5.3 图片轮播
- 支持手势滑动切换
- 支持按钮点击切换
- 自动同步当前图片索引
